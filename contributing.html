Contributing

Issues

Information

On this page you can find information about how to contribute to the my website by creating bug reports or fixing them and creating pull-requests. You can also see latest bigger changes and contributors at the Revisions Page. More Information is below.

Grammar

Content should be grammatically correct and the spelling should be US English, e.g. Color not Colour.

Issues

Have you found an issue with the website? That is awesome! Create an issue on my Github Repository and I will get around to it as fast as possible! You can post anything from navigation bugs to spelling mistakes. I want to make sure this project works 100% at any time.

Pull Requests

If you know how to code and you are interested in helping out feel free to submit a pull request on my Github Repository. You can work on fixing an existing issue or fixing an issue you just discovered. Make sure that your fork is up to date before you create a pull request. If the pull request is good I will merge it!
